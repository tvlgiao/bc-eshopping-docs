# Release Notes

## 1.0.0 — Initial public release (May 2026)

- Brand-new eShopping layout: top bar + sticky main nav + sidebar + 4-zone footer
- 4 built-in variations: **Industrial**, **AutoParts**, **Electronics**, **Packaging** — selecting a variation applies its color palette; the AutoParts, Packaging, and Electronics variations also apply fonts, trust-strip text, newsletter copy, promo text, SEO copy, and cart-goal tiers (the Industrial variation only overrides badge, rating, and price colors)
- Drag-and-drop content via BigCommerce's built-in Page Builder and widget regions
- Quick-search keyword suggestions (CSV-driven autocomplete + inline completion)
- Frequently Bought Together bundles with optional discount
- Bulk-order mode for B2B catalogs
- Urgency strips on PDP (configurable ranges)
- Recent-sales toasts (configurable per page)
- Cart drawer with progress bar + suggested products
- 3 popup types: Newsletter, Promo, Exit-intent
- Mobile sticky Add-to-Cart on PDP

### Performance

- Optimized for Core Web Vitals (LCP, CLS, INP), with lazy-loaded below-fold imagery and skeleton placeholders to minimize layout shift

### Known limitations

- Recent-sales toasts show "Someone in {location} just purchased {product}" using real products (fetched live). Locations and time-ago labels are both optional; when a time is not specified, the theme generates a random "X minutes/hours/days ago" label at display time. These toasts do not reflect actual order data.
