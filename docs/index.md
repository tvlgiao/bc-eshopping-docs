## What you will learn

This guide is written for **store owners** — not developers. Follow it from top to bottom and you will end up with a storefront that **closely matches one of the eShopping demos**, with no coding required.

!!! note "A couple of demo sections need real data first"
    The **Best Sellers** slider is turned on in every demo but only appears once your store has top-seller sales data, so it stays hidden on a freshly imported store. See the [Home Page overview](usage/home-industrial.md) for details.

You will learn how to:

1. Install and activate the eShopping theme on your BigCommerce store.
2. Pick the variant (Industrial / AutoParts / Electronics / Packaging) that matches your products.
3. Import the matching demo products and widgets in a few clicks.
4. Customize the home page, product page, category page, cart, search, and blog through **Page Builder** — without touching any code.
5. Use the theme's own configurable sections (no app required) plus the **AI HTML Generator | PapaThemes** widget — provided by the PapaThemes app — to build any layout you need.

## The 4 demo stores

eShopping ships with four ready-made demo configurations. Each one is a full export of a real BigCommerce storefront — products, categories, navigation, hero banners, widget content, and theme colors. You can clone any one of them onto your own store using the same import tool.

Picking a variation in Page Builder applies that demo's visual settings in one click. The **Industrial** look is the theme's default; the **AutoParts**, **Electronics**, and **Packaging** variants additionally override the full palette, fonts, and marketing copy.

!!! tip "Not sure which one to pick?"
    Open all four demos in browser tabs, scroll through them, and pick the one **closest to the look you want**. You can switch variations later — your products and orders are never affected, only the visual settings change.
